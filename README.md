# DB Manager App

## Установка и запуск

```bash
git clone https://github.com/yourusername/db-manager-app.git
cd db-manager-app
docker-compose up --build
```

## Возможности

- Поддержка PostgreSQL, MySQL, MongoDB
- CRUD интерфейс
- Визуализация структуры БД
- Аутентификация и роли пользователей
- Импорт/экспорт данных
